# Library API Final Project

## Description
This API is for managing a library system, including users, books, and reservations.

## Features
* Authentication for Users and Librarians
* Book Management
* Reservations System

## Setup
1.  `npm install`
2.  `npm start`

## API Endpoints
| POST /api/auth/register
| POST /api/auth/login|
| GET /api/books
| POST /api/books
| POST /api/reservations
